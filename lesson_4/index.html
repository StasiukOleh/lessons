Властивості CSS для роботи з текстом

Оформлення тексту-одна з ключових частин веб-дизайну.CSS надає широкий набір інструментів для керування шрифтами,розміром,відступами та випівнюванням.

Шрифт і його параметри

font-family - визначає шрифт тексту

font-size - задає розмір тексту

font-weight - контролює товщину тексту

значення font-weight: 100.

значення font-weight: 200.

значення font-weight: 300.

значення font-weight: 400.

значення font-weight: 500.

значення font-weight: 600.

значення font-weight: 700.

значення font-weight: 800.

значення font-weight: 900.

font-style - задає стиль(звичайний або курсив)

Вирівнювання і відступи

text-align - вирівнює текст(приклад заголовок по центу)

line-height - задає висоту рядка

letter-spacing - відстань між буквами

word-spacing - відстань між словами

Декорації тексту

text-decoration - додає підкреслення зверху

text-decoration - додає підкреслення знизу

text-decoration - додає підкреслення хвилясте

text-decoration - додає підкреслення точково

text-transform - змінює регістр тексту

text-indent - відступ першого рядка

Робота з кольорами в CSS

Кольори використовуються для тексту,фону,рамок і т.д.

Властивість color(задає колір тексту)

Формати кольорів(CSS підтримує кілька форматів кольорів)

Назви кольорів(color: blue;)

HEX - шістнадцятковий формат ( color: #ff0000;)

RGB - color: rgb(255, 0, 0);

RGBA - з прозорістю color: rgba(255, 0, 0, 0.5);

HSL - color: hsl(0, 100%, 50%);

Тіні в CSS

Тіні додають глибину та стиль елементам

text-shadow - тінь тексту(додає тінь до тексту)

Висновок

CSS надає потужні інструменти для оформлення тексту,роботи з кольорами та створення візуальних ефектів через тіні.Ти можеш створювати сучасні,красиві та зручні інтерфейси.
